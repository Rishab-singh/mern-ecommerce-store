# MERN E-commerce Store

A full-stack E-commerce platform built with the MERN stack.

## Features

- User Authentication (JWT)
- Email Verification
- Product Search & Filtering
- Product Reviews & Ratings
- Shopping Cart
- Secure Checkout
- Order Tracking
- Admin Dashboard
- Product Management
- User Management

## Tech Stack

Frontend
- React
- Tailwind CSS
- Axios
- React Router

Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## Project Structure

client/ → React frontend  
server/ → Express backend

## Installation

Clone the repository

Install dependencies
cd client
npm install

cd ../server
npm install

Run project

npm run dev
